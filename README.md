# MultiTask-UACM

Multitask-UACM es un gestor de operaciones genéricas multi-tarea con interfaz grafica de usuario

**Identificador del sistema** "mx.edu.uacm.is.slt.ds.multitask_uacm"

**Version actual** 1.0.0

## 📂 Estructura del Proyecto
El proyecto sigue el estándar de separación de responsabilidades:
* `src/main/java/`: Lógica del negocio (Modelos, Controladores, Interfaces).
* `src/main/resources/`: Recursos gráficos, archivos FXML y hojas de estilo CSS.
